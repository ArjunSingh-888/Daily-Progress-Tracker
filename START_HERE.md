## ⚡ Super Quick Start

### Windows Users
```
1. Double-click: start.bat
```

## 🎯 First Steps in the App

### 1. Add Your First Goal (30 seconds)
1. Click the **"+ Add Goal"** button
2. Type your goal (e.g., "Exercise for 30 minutes")
3. Set weight (1-10, or leave as 1)
4. Click **"Add Goal"**

### 2. Track Your Progress (10 seconds)
1. Click the checkbox next to your goal when done
2. Watch the progress circle update
3. Feel accomplished! ✅

### 3. Review Your History (1 minute)
1. Click **"History"** in the bottom navigation
2. See your completed days
3. Click any date to view details

### 4. Customize Settings (30 seconds)
1. Click **"Settings"** in the bottom navigation
2. Toggle **Dark Mode** for night use
3. Export your data as backup

## 🎨 Understanding Goal Weights

```
Weight 1 = Regular task
Weight 2 = Important task  
Weight 3 = Very important task
Weight 4-10 = Critical tasks

Example:
- Exercise (Weight 2)
- Read (Weight 1)  
- Work Project (Weight 3)

Total weight = 6
Complete Exercise = 33% progress!
Complete Work Project = 50% progress!
```

## 🆘 Quick Troubleshooting

### App Won't Start?
```bash
# Delete and reinstall
rm -rf node_modules
npm install
npm run dev
```

### Data Not Saving?
- Don't use private/incognito mode
- Check browser allows localStorage
- Try different browser

### Can't Install as PWA?
- Must use HTTPS (or localhost is OK)
- Clear browser cache
- Try different browser
- Check [SETUP_GUIDE.md](SETUP_GUIDE.md#troubleshooting)

### Something Else?
→ Check [SETUP_GUIDE.md](SETUP_GUIDE.md) Troubleshooting section
